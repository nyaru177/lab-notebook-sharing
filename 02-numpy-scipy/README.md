# 02 NumPy / SciPy

对应《实验室新生暑期居家集训学习计划》Week 1 的 NumPy 核心，SciPy 为选学补充。

可分享主题（例如，以下内容仅供参考，不限于这些内容）：

- 数组创建、属性、数据类型
- 索引、切片、布尔索引、花式索引
- 广播机制
- 矩阵运算与向量化
- z-score、余弦相似度、softmax、one-hot、k-NN
- SciPy 统计检验、距离计算、信号处理等选学内容
