# Notebook 模板

`notebook_template.ipynb` 是共享仓库的结构兜底。提交新内容时，优先打开各分类目录 `example/` 子目录中的 `*_example.ipynb`，理解一份真实 Notebook 应该怎样组织；需要空白起点时，再复制模板到自己的姓名文件夹中并替换占位内容。

使用时注意：

1. 把标题、作者、周次、分类、关键词、学习目标和运行环境补完整。
2. 保留“背景、最小示例、关键解释、实验或练习、总结与参考”这些基本结构。
3. 删除不适用的占位文字，并根据主题补充必要的 Markdown 和代码。
4. 从头运行一次后再提交；模板中的示例代码不能代替自己的核心实验。

主题不需要局限于分类页列出的示例，只要和集训目标或科研实践相关即可。
