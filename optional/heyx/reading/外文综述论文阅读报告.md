## 论文阅读报告

**标题**：Video Anomaly Detection in 10 Years: A Survey and Outlook  
**发表**：Neural Computing and Applications，2025；公开版：arXiv:2405.19387  
**一句话总结**：这篇综述回顾近十年深度学习视频异常检测，重点围绕特征提取、学习范式、损失函数、异常评分和视觉语言模型总结技术演进。

### 1. 研究背景与动机

- 解决的核心问题是：VAD 在安防、医疗、环境监测等场景中很重要，但近十年方法从传统深度模型扩展到弱监督、自监督、无监督和视觉语言模型，已有综述对这些新趋势覆盖不够细。
- 现有方法的不足在于：异常样本稀缺、标注成本高、异常定义模糊，且很多模型只关注网络结构，缺少对数据集、特征、损失函数、正则约束和异常分数的综合分析。
- 该问题重要，是因为真实场景中的异常具有开放性和长尾性，模型不仅要在标准数据集上高 AUC，还要能解释、泛化并适应复杂场景。
  行文思路：

1. **问题引入**：说明 VAD 在监控、医疗、环境监测等场景中的应用价值。
2. **十年变化背景**：指出深度学习、CNN、ViT、VLM 推动 VAD 从传统特征走向深层表征。
3. **研究方法说明**：声明聚焦过去十年、顶级 CV 会议/期刊和 50+ 方法。
4. **相关综述回顾**：说明已有综述未充分覆盖 VLM、损失函数、正则化、异常分数等关键模块。
5. **问题形式化**：给出视频级异常分数、阈值判定、预测标签等基本定义。
6. **系统分析框架**：按 datasets、feature extraction、learning/supervision、loss、regularization、anomaly score 逐节展开。
7. **评估和SOTA比较**：整理常见数据集、AUC/AP/EER 指标，并比较 2013-2023 代表方法。
8. **文献计量分析**：用 VOSviewer 展示 2023-2024 近作主题聚类。
9. **未来方向**：强调 VLM、文本特征、多样化数据集、鲁棒评估和开放场景。

### 2. 主要方法

- 核心思路：论文以近十年深度 VAD 为主线，按监督、弱监督、自监督、无监督等学习范式组织方法，并横向比较特征提取、损失函数、时空正则化、异常分数预测等关键模块。
- 方法架构图理解：可以理解为“输入视频 -> 特征提取器 -> 学习范式/损失函数 -> 异常评分 -> 评价与展望”的流程。论文特别强调视觉语言模型可以把文本语义引入异常理解。
- 关键公式理解：该文是综述，不提出单一公式。关键思想是异常分数由模型对时空模式的偏离程度、分类/排序结果或视觉-语言语义不一致性共同决定。

### 3. 实验设置

- 数据集：重点讨论 UCF-Crime、ShanghaiTech 等公开基准，并提到大型训练数据、异常类别多样性和文本描述数据对未来 VAD 的影响。
- 评估指标：主要关注 AUC，也涉及不同模型在基准数据集上的定量对比。
- 对比方法：覆盖传统深度学习模型、CNN、Autoencoder、GAN、Vision Transformer、弱监督 MIL、视觉语言模型等代表路线。

### 4. 主要结论与核心数值

- 论文指出 2017 至 2023 年间，UCF-Crime 和 ShanghaiTech 上代表性方法的 AUC 持续提升，视觉语言模型带来了较明显的新增长点。
- 最重要结论是：VAD 研究正在从“仅学习视觉异常模式”转向“视觉、文本、语义共同参与的异常理解”。
- 对写作价值而言，它适合放在“近十年技术演进”和“大模型/视觉语言模型趋势”部分。

### 5. 我的疑问与不足

- 我没完全看懂的地方是：视觉语言模型用于 VAD 时，文本提示、视频片段语义和异常标签之间如何形成稳定对应关系。
- 局限性：综述覆盖范围较广，但对具体 VLM 方法的实验细节和推理成本讨论还可以更深入。
- 可能改进方向：构建带帧级文本描述的 VAD 数据集，设计能同时处理视觉异常和语义异常的损失函数。

## 论文阅读报告

**标题**：Deep Learning for Video Anomaly Detection: A Review  
**发表**：IEEE Transactions on Neural Networks and Learning Systems，2026  
**一句话总结**：这是一篇面向深度学习视频异常检测的综合综述，最有价值之处是把 VAD 按监督信号统一划分为半监督、弱监督、全监督、无监督和开放集监督，并补充了大模型、数据集、指标和开源资源。

### 1. 研究背景与动机

- 解决的核心问题是：视频异常检测研究路线很多，但已有综述多集中在传统的半监督/无监督范式，难以覆盖近几年弱监督、全监督、开放集监督和大模型方法的快速发展。
- 现有方法和综述的不足在于：分类标准不统一，有的按输入模态分，有的按重建/预测等任务分，导致不同监督设定之间的联系不清楚；同时对新兴的大模型 VAD、开放集 VAD、可解释 VAD 关注不足。
- 这个问题重要，是因为 VAD 在安防、交通、内容审核、工业监控等场景中有实际部署需求，而异常样本稀缺、场景变化大、误报成本高，研究者需要一个清晰框架来选择数据、模型和评价指标。
  行文思路：

1. **问题引入**：从 anomaly detection 的广泛应用切入，说明 VAD 在智能安防、内容审核、交通异常、自动驾驶等场景中的意义。
2. **深度学习时代的必要性**：通过论文数量增长和基准性能提升说明 VAD 已进入深度学习驱动阶段，传统综述已不能覆盖新范式。
3. **现有综述不足**：指出以往综述多集中于半监督/单场景/传统小模型，对弱监督、全监督、无监督、开放集、大模型和可解释 VAD 覆盖不足。
4. **统一任务定义**：用训练样本、异常样本、标签粒度和测试分布形式化区分五类 VAD。
5. **分任务展开**：依次讨论半监督、弱监督、全监督、无监督、开放集监督 VAD。
6. **面向未来**：提出综合基准、开放世界任务、预训练大模型、可解释 VAD 四个未来方向。

### 2. 主要方法

- 核心思路：论文不是提出单个检测模型，而是提出一个“综述式分类框架”。先用监督信号定义 VAD 任务，再分别讨论每类任务中常见的模型输入、网络结构、学习目标、优化方式和异常分数输出。
- 方法架构图理解：文中的核心图可以理解为“监督信号 -> 任务范式 -> 模型设计维度”的三层结构。监督信号决定训练数据里是否有正常/异常样本和标签粒度；任务范式决定模型学习正常性、视频级异常性、像素/帧级异常性还是未知异常；模型设计维度再落到输入、架构、损失和输出。

#### 一级分类：按监督信号划分VAD任务

| 一级任务范式            | 训练数据设定       | 标签粒度         | 核心目标                           | 主要优点                       | 主要局限                         |
| ----------------------- | ------------------ | ---------------- | ---------------------------------- | ------------------------------ | -------------------------------- |
| Semi-supervised VAD     | 仅正常视频         | 无异常标签       | 学习正常模式，测试时偏离正常即异常 | 不需采集异常样本               | 易把未见正常模式误判为异常       |
| Weakly supervised VAD   | 正常+异常视频      | 视频级标签       | 从粗标签中定位异常片段             | 标注成本低，适合大规模监控视频 | MIL 容易盲猜、伪标签噪声明显     |
| Fully supervised VAD    | 正常+异常视频      | 帧级/片段级精标  | 标准分类或定位                     | 性能上限高                     | 精细标注成本高                   |
| Unsupervised VAD        | 完全无标签测试视频 | 无标签           | 直接从未标注数据中发现异常         | 无需人工标注，可持续适配       | 性能和稳定性较弱                 |
| Open-set supervised VAD | 训练含已知异常类   | 测试含未知异常类 | 检测 seen/unseen anomalies         | 更贴近开放世界                 | 需要专门的开放集判别或未知类建模 |

#### 半监督VAD内部分类

半监督部分采用“五维框架”组织：

| 维度                 | 子类                                                      | 代表技术含义                                   |
| -------------------- | --------------------------------------------------------- | ---------------------------------------------- |
| Model Input          | RGB、Optical Flow、Skeleton、Hybrid                       | 从外观、运动、人体姿态、多模态输入角度组织方法 |
| Methodology          | Self-supervised、One-class、Interpretable learning        | 按正常模式学习机制划分                         |
| Network Architecture | 2D/3D CNN、LSTM/RNN、AE、GAN、GCN、Transformer、Diffusion | 按网络结构和生成机制划分                       |
| Refinement Strategy  | 伪异常、记忆库、特征约束等                                | 解决正常性泛化、背景干扰、异常稀缺             |
| Model Output         | Frame-level、Pixel-level、Patch-level、Object-level       | 按异常定位粒度划分                             |

####  弱监督VAD内部分类

弱监督部分同样不是单纯按 MIL 罗列，而是拆成四个方面：

| 维度                | 子类                                                         | 代表技术含义                                                 |
| ------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| Model Input         | RGB、Optical Flow、Audio、Text、Hybrid                       | 输入通常是预训练模型提取的特征，如 C3D、I3D、TSN、VideoSwin、CLIP、VGGish |
| Methodology         | One-stage MIL、Two-stage self-training                       | 前者直接用 MIL 排序/TopK，后者先训练初始模型再生成伪标签自训练 |
| Refinement Strategy | Temporal modeling、Spatio-temporal modeling、Modified MIL、Metric learning、Knowledge distillation、Large models | 用于缓解视频级弱标签带来的定位不确定性                       |
| Model Output        | Frame-level、Pixel-level                                     | 从异常分数扩展到更细粒度定位                                 |

####  开放集与大模型相关位置

这篇综述把大模型放在未来趋势与弱监督/开放集相关方法中处理，而不是单独建成 VAU 体系。它强调：

- CLIP/VLM 可提升特征表达、语义理解和零样本能力；
- LLM 可用于生成异常解释
- Open-vocabulary learning 是通向 open-world VAD 的关键路径；
- 可解释 VAD 需要结合输入层对象/轨迹、算法层知识图谱/意图预测/LLM 推理，以及输出层时空解释。
- 关键公式理解：论文给出统一优化形式 `l = L(Phi(theta, X), X or Y)`。其中 `Phi` 是检测模型，`X` 是视频样本，`Y` 是监督标签；当没有标签时，损失更多依赖样本自身，例如重建或预测误差；有标签时，损失转向分类、排序或多实例学习目标。

### 3. 实验设置

- 数据集：论文整理了常用 VAD 数据集，例如 UCSD Ped1/Ped2、CUHK Avenue、ShanghaiTech、UCF-Crime、XD-Violence、UBnormal 等，并覆盖不同监督设置。
- 评估指标：半监督和无监督常用帧级 AUC、EER 等；弱监督常用 AUC、AP；更细粒度任务可能需要时序定位、目标级定位或像素级定位指标。
- 对比方法：作为综述，论文对比的是不同范式下的代表性方法，而不是只对比一个模型的 baseline。它关注各类方法的性能趋势、输入差异、监督成本和适用场景。

### 4. 主要结论与核心数值

- 论文指出 VAD 性能持续上升，例如 CUHK Avenue 上半监督方法从早期约 70.2% AUC 提升到约 90.1% AUC，说明深度模型仍有明显发展空间。
- 最重要结论是：VAD 不能再只被理解为“用正常视频训练、用重建误差找异常”的单一路线。弱监督、全监督、开放集监督和大模型方法正在共同扩展 VAD 的问题边界。
- 对阅读用途而言，这篇最适合放在“国内外研究现状”的开头，用来搭建总框架。

### 5. 我的疑问与不足

- 我还需要进一步细读的是：开放集监督 VAD 与开放词汇 VAD、大模型 VAD 之间的边界如何划分，尤其是未知异常的评价协议是否足够统一。
- 局限性：综述很全面，但因为覆盖面大，每个具体方向的技术细节不会特别深；部分性能比较受数据划分、特征提取器和评价粒度影响，横向比较时要谨慎。
- 可能改进方向：后续可以围绕“统一开放集评测基准”“大模型提示与视频时序建模结合”“可解释异常定位”继续展开。
