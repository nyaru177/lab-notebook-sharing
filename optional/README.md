# Optional

这里放跨多个技能分类、后续周次或对科研有帮助的补充 Notebook。Week 6-8 的示例也放在这里，作为论文精读、代码复现和优化实验的提交参考。

可选主题（例如，以下内容仅供参考，不限于这些内容）：

- Docker 环境封装
- LoRA / QLoRA
- 实验配置管理
- 随机种子与可复现性
- 日志记录与实验追踪
- 分享推荐的工具，工作流

当前周次示例：

- `example/week6_paper_reading_example.ipynb`
- `example/week7_reproduction_scaffold_example.ipynb`
- `example/week8_optimization_example.ipynb`

## 提交位置

请在本目录下新建自己的姓名文件夹，再把 Notebook、图片、依赖和说明文件放入其中；不要直接把个人文件放在模块根目录。仓库示例统一放在 `example/` 文件夹。

注意：这些内容不作为前三周公共 Notebook 的必修分类。
